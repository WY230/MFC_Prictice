# MFC_Prictice

本仓库为本人在研究生阶段使用C++和MFC完成的练习项目合集。

每个项目均包含独立的‘README.md’文件，详细说明项目功能、界面与技术实现，可点击对应子目录查看详情。

## 开发环境
- **编程语言**:C++
- **开发平台**:Visual Studio 2022
- **框架**:MFC
- **操作系统**:Windows 11

- 强化事件响应、消息映射、输入处理等实用能力
